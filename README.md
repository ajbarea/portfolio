## Development

### Start hot-reloading development server (<http://localhost:3000/>)

```bash
npm install
npm audit fix
npx update-browserslist-db@latest
npm run dev
```

> **Note:** Deployment to GitHub Pages is automated via GitHub Actions workflow on push to `main` branch.
